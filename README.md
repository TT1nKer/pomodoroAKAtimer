# pomodoroAKAtimer

minimalist pomodoro timer. single-file html, runs offline in any browser.

## use

open `timer.html` in a browser. press `f` for fullscreen.

## features

- custom session sequences (drag to reorder) + presets: classic / long / ultradian / sprint
- click digits to edit time, or scroll / arrow keys to ±1m
- focus stats — daily, weekly, monthly, yearly + last-30-days chart
- end-of-phase alert with three intensities: gentle / strong / persistent
- wake-lock keeps the screen awake while running
- dynamic progress favicon, "ends at hh:mm" eta
- fully offline — vue 3 bundled, no network calls

## keys

`space` start/pause · `n` next · `r` reset · `f` fullscreen · `e` edit time · `s` session · `,` settings · `↑`/`↓` ±1m (`shift` for ±5m)
